A. PL/0 语言是 Pascal 语言的子集
   - 数据类型只有整型
   - 标识符的有效长度是 10 ，以字母开头的字母数字串
   - 数最多 14 位
   - 过程无参，可嵌套（最多三层），可递归调用
   - 变量的作用域同 Pascal ，常量为全局的

B. 语句类型：
   - 赋值语句： if…then…, while…do…, read, write, call
   - 复合语句： begin …end
   - 说明语句： const…, var…, procedure…

C. 13个保留字：
   - If/then, while/do, read/write,
   -  call, begin/end, const, var, procedure, odd
